
# App Dino Tv - Clone Netflix - Ionic

Neste projeto, utilizei o Angular e Ionic.

# Informações:

    • API do projeto foi pega do site https://themoviedb.org/
    • Todo o direito de imagens são da Netflix


# Instalação:

    • npm install
    
    • npm i -g cordova

    • ionic serve -b
